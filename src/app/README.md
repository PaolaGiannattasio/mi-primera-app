# Mi Primera App Angular

## Descripción

Proyecto realizado con Angular como práctica inicial del curso "Conociendo Angular".

La aplicación incluye:

- Cambio del título principal.
- Variables en el componente principal.
- Interpolación de datos en HTML.
- Imagen agregada desde la carpeta assets.

---

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS

---

## Instalación

Clonar el repositorio:

git clone URL_DEL_REPOSITORIO

Instalar dependencias:

npm install

Ejecutar la aplicación:

ng serve

Abrir en el navegador:

http://localhost:4200

---

## Estructura principal del proyecto
Nota: La versión actual de Angular utilizada genera proyectos con arquitectura standalone, por lo que algunos archivos tradicionales como app.module.ts no se crean automáticamente.

### src/app

Carpeta principal donde se encuentran los componentes y la lógica de la aplicación.

### app.ts

Archivo TypeScript del componente principal. Contiene variables y lógica de la aplicación.

### app.html

Archivo HTML principal donde se muestra el contenido visual de la aplicación.

### assets/

Carpeta utilizada para almacenar recursos estáticos como imágenes e íconos.

### environments/

Carpeta utilizada para configuraciones de distintos entornos (desarrollo y producción).

---

## Funcionalidades implementadas

- Cambio del título principal.
- Agregado de nombre y motivo para aprender Angular.
- Uso de interpolación mediante {{ }}.
- Actualización automática de datos en pantalla.
- Incorporación de imagen desde assets.
